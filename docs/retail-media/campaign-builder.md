# Campaign Builder

Create and manage Retail Media campaigns.

## Campaign Structure
A campaign consists of four main components, validated by the system:

| Component | Required | Description |
|-----------|----------|-------------|
| **Vendor** | **Yes** | The brand or advertiser funding the campaign. |
| **Campaign**| **Yes** | General settings (Dates, Budget, Name). |
| **Audience**| **Yes** | Targeting parameters (Demographics, Location). |
| **Creative**| **Conditional**| The visual assets (Banner, Video). Required for non-MDI campaigns. |

## Campaign Types
- **Standard**: Display ads.
- **MDI**: Managed Digital Integration (Creative handled externally).
