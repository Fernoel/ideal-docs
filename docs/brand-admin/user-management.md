# User Management

Manage system access for your organization.

## Roles & Permissions
- **Admin**: Full system access.
- **Content Manager**: Can edit circulars and assets but cannot change system settings.
- **Store Manager**: Access limited to specific store nodes.
- **Viewer**: Read-only access.

## Actions
- **Invite User**: Send an email invitation to join the platform.
- **Revoke Access**: Immediately disable a user's login.
