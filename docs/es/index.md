## Introducción General

La plataforma Ideal Sale, desarrollada por Design House, es una herramienta para gestionar contenido digital, como volantes promocionales. Incluye varios módulos para crear, organizar y publicar contenido relacionado con marcas, tiendas y promociones.

Este manual se centra en el módulo **Circular Builder**, que permite a los usuarios crear y gestionar elementos de contenido individuales o grupos de contenido para volantes digitales.

    Este es un borrador inicial basado en la información proporcionada. Asegúrese de tener permisos de administrador para acceder al módulo (por ejemplo, a través de la URL de administración como se muestra en las capturas de pantalla).
