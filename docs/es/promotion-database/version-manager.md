# Gestor de Versiones

El Gestor de Versiones de la Base de Datos de Promociones maneja el ciclo de vida de sus datos promocionales.

## Funcionalidad
- **Versionado**: Rastree los cambios en las promociones a lo largo del tiempo.
- **Reversión (Rollback)**: Restaure versiones anteriores de datos de promoción si ocurren errores.
- **Registro de Auditoría**: Vea quién cambió qué y cuándo.
- **Estado de Sincronización**: Supervise la sincronización con sistemas de precios externos.
