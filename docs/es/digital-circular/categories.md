# Categorías

Organice sus productos en grupos lógicos para una mejor navegación.

## Campos
- **Nombre**: Nombre para mostrar (ej. "Lácteos", "Frutas y Verduras").
- **Categoría Padre**: Para crear subcategorías (jerarquía).
- **Orden**: Determina la secuencia de visualización en el menú.
- **Icono**: Icono opcional para representar la categoría.

## Uso
Las categorías se utilizan en el **Constructor de Circulares** para filtrar productos y en el **Frontend** para generar el menú de navegación.
