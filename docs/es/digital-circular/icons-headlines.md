# Iconos y Titulares

Gestione elementos gráficos reutilizables.

## Iconos
Gráficos vectoriales pequeños (SVG) o imágenes utilizadas para categorías, insignias (ej. "Sin Gluten") y elementos de la interfaz.

## Titulares
Gráficos de texto estilizados utilizados como encabezados de sección en el circular (ej. "Ahorros Semanales").
