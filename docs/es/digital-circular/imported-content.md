# Contenido Importado

Gestione fuentes de datos externas (ej. sistemas POS, ERPs).

## Formatos Soportados
- **CSV**: Valores separados por comas estándar.
- **XML**: Fuentes de datos estructurados.
- **Integración API**: Sincronización en tiempo real.

## Mapeo
Configure cómo los campos externos se asignan a los campos internos del sistema Ideal (ej. mapear "UPC" a "Código de Barras").
