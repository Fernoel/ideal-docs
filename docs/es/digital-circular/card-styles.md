# Estilos de Tarjeta

Defina la apariencia visual de las tarjetas de productos.

## Opciones de Personalización
- **Borde**: Color, ancho, radio.
- **Sombra**: Intensidad de la sombra paralela.
- **Tipografía**: Familia de fuente, tamaño y peso para precio y descripción.
- **Diseño de Precio**: Posición del signo de dólar, centavos y unidad.
- **Fondo**: Color o patrón.

## Uso
Aplique diferentes estilos para resaltar ofertas específicas (ej. estilo "Compra Caliente" vs estilo "Estándar").
