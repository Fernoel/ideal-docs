# Gestión de Usuarios

Gestione el acceso al sistema para su organización.

## Roles y Permisos
- **Admin**: Acceso completo al sistema.
- **Content Manager**: Puede editar circulares y activos pero no cambiar la configuración del sistema.
- **Store Manager**: Acceso limitado a nodos de tienda específicos.
- **Viewer**: Acceso de solo lectura.

## Acciones
- **Invitar Usuario**: Envíe una invitación por correo electrónico para unirse a la plataforma.
- **Revocar Acceso**: Deshabilite inmediatamente el inicio de sesión de un usuario.
