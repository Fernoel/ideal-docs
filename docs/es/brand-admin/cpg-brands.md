# Marcas CPG

Gestione las relaciones con Bienes de Consumo Envasados (CPG).

## Funcionalidad
- **Perfiles de Marca**: Logotipo, descripción e información de contacto para socios CPG.
- **Intercambio de Activos**: Permita que las marcas CPG carguen activos directamente a su biblioteca.
- **Informes**: Comparta métricas de rendimiento con socios CPG.
