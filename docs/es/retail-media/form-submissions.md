# Envíos de Formularios

Vea y gestione los datos enviados a través de formularios de Retail Media (ej. sorteos, suscripciones a boletines).

## Características
- **Cuadrícula de Datos**: Vea los envíos en formato tabular.
- **Exportar**: Descargue los envíos como CSV/Excel.
- **Webhooks**: Configure el reenvío de datos en tiempo real a CRMs.
