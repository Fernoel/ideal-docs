# Categories

Organize your products into logical groups for better navigation.

## Fields
- **Name**: Display name (e.g., "Dairy", "Produce").
- **Parent Category**: For creating sub-categories (hierarchy).
- **Sort Order**: Determines the display sequence in the menu.
- **Icon**: Optional icon to represent the category.

## Usage
Categories are used in the **Circular Builder** to filter products and in the **Frontend** to generate the navigation menu.
